# 메인 차별점

여행에서 이동경로를 보여주는 sns

- 지도 그림에서 이동경로를 보여줌
- 인스타그램의 경우 올리는 사람 중심
- day trip의 경우 보는사람의 중심으로 만들었음
- 우리는 올리는 사람과 보는 사람 모두를 만족할 수 있는 서비스


- 유저 시나리오 :
    - 첫페이지에서 게시물은 랜덤으로 보여줌
    - 위치로 검색을 하면 틱톡, 쇼츠처럼 위 페이지들을 보여줌 아래로 내리면 다음 개시물을 보여줌
        - 게시글 순서같은경우
            - 새로고침하면 랜덤으로 섞임
        - 위치 검색같은 경우 현재 위치(동단위로)
        - 북마크
    - 만약에 마음에 들면 해당 페이즈를 클릭하면 블로그형식으로 아래로 내리면 위치1(그림,설명), 위치2(그림,설명)… 이렇게 나옴
        - 북마크 있음
        - 댓글기능있음
        - 댓글기능 옆에 좋아요 클릭
    - 블로그나 게시물에서 해당 게시글을 올린 사용자를 클릭하면 해당 사용자가 올린 경유지를 텍스트로만 리스트로 보여줌
    - 로그인기능
        - 댓글
        - 좋아요
        - 북마크
